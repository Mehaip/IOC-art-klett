# Romanian Version - Sa invatam planetele

## Summary of Changes

All text in the game has been translated to Romanian for use in Romanian schools.

### Main Menu
- Title: **"SA INVATAM PLANETELE"**
- Subtitle: **"(cu ajutorul manualului ArtKlett)"**
- Start Button: **"INCEPE"**

### Planet Names (Romanian)
1. **Mercur** (Mercury)
2. **Venus** (Venus)
3. **Pamant** (Earth)
4. **Marte** (Mars)
5. **Jupiter** (Jupiter)
6. **Saturn** (Saturn)
7. **Uranus** (Uranus)
8. **Neptun** (Neptune)

### UI Elements Translated
- Exploration instructions: "Foloseste sagetile pentru a te misca | SPACE pentru interactiune"
- Progress tracker: "Planete explorat–µ: X/8"
- Quiz prompt: "Apasa orice tasta pentru a continua la quiz..."
- Question counter: "Intrebarea X/5"
- Score display: "Scor: X/5"
- Continue prompts: "Apasa orice tasta pentru a continua..."

### Dodge Game
- Timer: "Timp: Xs"
- Instructions: "Foloseste sagetile STANGA/DREAPTA pentru a evita!"
- Success message: "FELICITARI! Ai evitat toti asteroizii!"
- Failure message: "LOVIT DE ASTEROID! Mai mult noroc data viitoare!"

### Educational Content
- All planet information (4 facts per planet) translated to Romanian
- All quiz questions (5 per planet, 40 total) translated to Romanian
- All answer options translated to Romanian

### Reference
The game mentions the **ArtKlett educational manual** in the subtitle as requested.

## Testing
The game has been tested and runs successfully with all Romanian text displayed correctly using PIL/Pillow fonts.
